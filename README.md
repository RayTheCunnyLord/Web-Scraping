# Web-Scraping
my own web-scraper!
- mainly using puppeteer
    https://pptr.dev/



how to install?

<code>git clone https://github.com/RayTheCunnyLord/Web-Scraping/tree/main</code> <br>
<code>npm install puppeteer</code> <br>
<code>npm install readline-sync</code>


how to run?

<code>node yt.js</code>
